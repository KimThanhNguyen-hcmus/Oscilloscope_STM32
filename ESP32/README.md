# ESP32-32S

This MCU will be responsible for recieve and analyze data sent by STM32. This part will include:

- **Wifi Connection**
- **HTTP & WebSocket Configuration**
- **Analyze Algorithm**
- **I2C & USART Configuration**
- **OLED Connection & Wave Drawing**
- **USART Packets Recieving**
